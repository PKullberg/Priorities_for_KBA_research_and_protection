These are folders contain setup files for zontion runs needed in the publication Using Key Biodiversity Areas to guide effective expansion of the global protected area network (Kullberg, di Minin and Moilanen). Runnig the analyses with these setup files requires linux system with Zonation 4 conservation prioritization software installed (). Runnig all the analyses requires  up to 50GB memory and will take several days. For further description of the variants see the reports/Priorities_for_KBA_research_and_protection.Rmd document.
